# Черновик (не используется)

Это незавершённый черновик backend'а на Java/Spring Boot — первоначальный
вариант технологии для «Карты скорби» до пересмотра решения.

По факту реальных условий сервера (shared-хостинг без root/Docker, без
установленной JVM) выбор был сознательно изменён на **Go** — см. рабочую
реализацию в `../server/` и обоснование в `../server/README.md`.

Здесь оставлен только `pom.xml` (дальше в реализацию не пошли, чтобы не
терять время на путь, который не будет использован) — на случай, если он
пригодится как отправная точка, если решение о технологии когда-нибудь
пересмотрят снова.
